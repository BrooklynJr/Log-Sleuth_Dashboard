# 🔍 Log-Sleuth Dashboard (v3.0)
**Developed by Brooklyn | IT Service Desk Automation**

### 🛠️ วิธีใช้งานสำหรับเพื่อนในทีม:
1. ดาวน์โหลดโฟลเดอร์นี้ลงเครื่อง
2. ติดตั้ง Library: `pip install -r requirements.txt`
3. เริ่มรันระบบ: `python -m streamlit run app.py`

### 🌟 ฟีเจอร์เด่น:
* **Smart Analysis:** แปล Error ID จาก Event Viewer เป็นภาษาไทย
* **One-Click Diagnostic:** เช็ค IP, เน็ตหน่วง, และสุขภาพแบตเตอรี่ในปุ่มเดียว
* **Pro-Active Support:** ช่วยวิเคราะห์ปัญหา Wi-Fi และ Domain Sync ได้ทันที
